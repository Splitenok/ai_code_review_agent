# ai_code_review_agent
Описание:     Мультиагентная система автоматического код-ревью для Python и SQL (PostgreSQL / MS SQL Server) с интеграцией в GitHub Pull Requests      
Идея:    Агент автоматически срабатывает при открытии PR, анализирует изменённые .py и .sql файлы и постит структурированный комментарий прямо в PR — как живой ревьюер
